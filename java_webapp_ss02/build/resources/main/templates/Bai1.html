Giới thiệu JPA

JPA (Java Persistence API) là một đặc tả (specification) của Java EE (nay là Jakarta EE) dùng để quản lý dữ liệu trong cơ sở dữ liệu quan hệ thông qua mô hình hướng đối tượng. JPA không phải là framework, mà là chuẩn chung để các thư viện ORM như Hibernate, EclipseLink, hay OpenJPA triển khai. Trong đó, Hibernate là implementation phổ biến nhất và là mặc định khi dùng JPA trong Spring Boot.

JPA cung cấp các annotation như @Entity, @Id, @OneToMany để ánh xạ giữa các class Java và bảng dữ liệu. Việc tích hợp JPA trong Spring Boot thông qua spring-boot-starter-data-jpa giúp giảm code xử lý database, dễ maintain, và tách biệt logic khỏi công nghệ cụ thể. Tóm lại, JPA giúp chuẩn hóa cách tương tác với CSDL trong Java, còn Hibernate giúp hiện thực hóa đặc tả đó một cách hiệu quả.
